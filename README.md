# hb-demo-platform

Hepsiburada benzeri, eğitim amaçlı bir e-ticaret demo platformu.

Bu proje ile uçtan uca şu konuları öğrenmek hedeflenmektedir:

- Git ve GitHub ile profesyonel proje yönetimi
- Python tabanlı mikroservis geliştirme
- Docker ve Docker Compose
- Ağ yapısı ve servisler arası iletişim
- CI/CD pipeline
- Kubernetes deployment
- Ingress, ConfigMap, Secret, health checks
- DevOps best practice'leri

## Hedef Mimari

Planlanan servisler:

- api-gateway
- product-service
- cart-service
- order-service
- user-service
- frontend
- postgres
- redis

## Klasör Yapısı

- `services/` : uygulama servisleri
- `infra/` : Docker, Compose, Kubernetes ve CI/CD dosyaları
- `docs/` : mimari notlar ve öğrenme dokümanları
- `scripts/` : yardımcı scriptler
